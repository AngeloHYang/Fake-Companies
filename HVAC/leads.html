The Agentic Economy

How Customer Agents Find Our Business Agent

Sarah's home agent knows to contact HVAC businesses in the area. It finds Mike's HVAC through a public agent directory, negotiates the service call, and reports back to Sarah — all without Sarah having to pick up the phone.

🔍

Agent Discovery

Sarah's agent searches a local HVAC agent directory filtered by service area, rating, and availability. It finds Mike's HVAC, confirms 24/7 availability, and initiates contact.

🤝

Capability Handshake

The two agents exchange capability profiles: service area, brands covered, pricing structure, availability windows, and escalation contacts. This happens before the first customer message.

💬

Negotiation Loop

Sarah's agent describes the problem. Mike's agent requests photos, looks up the model number, runs diagnostic probability scoring, and proposes pricing and timing. The loop ends with a confirmed booking.

📬

Human-in-the-Loop

Both humans receive a notification. Sarah's agent sends her a summary: "Mike's HVAC is coming at 7am, $120 diagnostic fee, likely $280–$450 repair." Mike sees the lead card. Both can override before morning.
