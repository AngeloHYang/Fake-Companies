Screen 4a
Voice Mode — The Workhorse
Mike holds the button and talks naturally — no forms, no menus. The waveform visualizes listening. The live transcript shows what the agent heard in real-time. He can add a photo mid-call or pause if he needs to talk to the customer.

This is the primary interaction for a field tech. Everything else in the app is a result of what was said here.
Screen 4b
Agent Fans Out — 4 Tasks in Parallel
The moment Mike releases the button, the agent fans out to four external agents simultaneously. No sequential calls. No waiting. Grainger, Ferguson, the city building department, and the job history database — all contacted at once.

Each task is a live card. Mike can tap any card to see the agent-to-agent conversation happening in real time. He can intervene or override at any point.
Screen 5
Background Task Expanded — Pricing
Mike taps "Pricing furnace" and sees the result of the agent's conversation with two suppliers. Two prices, two availability windows, side by side. The agent makes a recommendation with reasoning — faster pickup, lower cost.

Mike taps Approve. The Ferguson order is placed. The agent updates the draft quote automatically and sends an ETA update to Sarah.
What Mike Said

One Voice Instruction. Four Agent Calls.

This is the agentic economy in action. One sentence from Mike turns into a cascade of parallel agent calls — all happening while he's still standing in Sarah's basement.

Mike's voice instruction
"Okay I'm at Sarah's place. The furnace is worse than I thought. Heat exchanger is cracked. We need a full replacement. Get me a quote on a Carrier 96K BTU from Grainger and Ferguson, check if we need a permit, and pull up her service history."
Agent immediately triggered 4 parallel tasks →
Pricing: Carrier 96K BTU → Grainger agent + Ferguson agent →
Permit check → City of Austin building dept →
Customer history lookup → 142 Oak St · prior visit 2022 →
Quote draft → assembling line items from supplier responses →
⚡

Parallel Execution

All four tasks run simultaneously. Total time: under 45 seconds from voice release to all results populated.

🔍

Live Visibility

Mike sees each task as a spinner card. He can tap any of them mid-execution to see what the agents are saying in real time.

🧑‍💼

Human Override

Every task has an Approve/Override button. Mike is always in control — the agent makes recommendations, Mike makes decisions.

📋

Auto-Quote Build

As pricing comes in from suppliers, the quote draft populates automatically. Mike reviews, taps Send, and the customer gets it instantly.
