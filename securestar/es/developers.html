Inicio Rápido

Obtén tu primera calificación de riesgo en menos de 5 minutos. Necesitarás una clave API de tu panel de SecureStar.

Instala el SDK

Instala el SDK de Python usando pip. También están disponibles los SDKs de Node.js y Java.

bash
pip install securestar-sdk

Configura tu clave API

Establece tu clave API como una variable de entorno o pásala directamente al cliente.

python
import securestar

client = securestar.Client(
  api_key="ss_live_..."
)

Califica tu primera transacción

Llama al método score() con los datos de la transacción. La respuesta incluye una calificación de riesgo, decisión y explicación.

python
result = client.score({
  "transaction_id": "txn_9x2q8a",
  "amount": 4820.00,
  "currency": "USD",
  "account_id": "acct_h82jk",
  "merchant_id": "merch_4x9m",
  "ip_address": "104.22.1.84",
  "device_id": "dev_8k2l",
})

print(result.risk_score) # 0.08
print(result.decision) # "approve"
print(result.latency_ms) # 43

POST /v2/score

Califica una sola transacción en tiempo real. Devuelve una calificación de riesgo, recomendación de decisión, explicación e ID de auditoría.

POST https://api.securestar.io/v2/score Calificación de transacciones en tiempo real

Parámetros de Solicitud

ParámetroTipoRequeridoDescripción
transaction_id string Requerido Tu identificador único de transacción
amount number Requerido Monto de la transacción en la moneda especificada
currency string Requerido Código de moneda ISO 4217 (e.g. USD)
account_id string Requerido Tu identificador interno de cuenta o usuario
merchant_id string Opcional Identificador del comerciante para señales de riesgo del comerciante
ip_address string Opcional Dirección IP del cliente para señales geográficas y de proxy
device_id string Opcional Huella digital del dispositivo para señales de nuevo dispositivo
metadata object Opcional Pares clave-valor arbitrarios incluidos en el registro de auditoría
💡 Consejo de latencia: Proporciona todos los campos opcionales disponibles. Cada señal adicional reduce la incertidumbre en el modelo y puede mejorar la precisión hasta en un 8%. La sobrecarga de latencia de campos adicionales es inferior a 2ms.

Conector de Kafka

El conector de Kafka de SecureStar se suscribe a tu tema de transacciones y publica eventos calificados en un tema de resultados, todo sin modificar tus productores existentes.

⚡ Latencia: El conector de Kafka añade una mediana de 8ms al camino de consumidor a consumidor de Kafka. Esto es típicamente más rápido que la API REST debido a la eliminación de la sobrecarga HTTP.
yaml — securestar-kafka.yaml
connector:
  input_topic: payments.transactions.raw
  output_topic: payments.transactions.scored
  error_topic: payments.transactions.errors
  consumer_group: securestar-scorer
  parallelism: 8

mapping:
  transaction_id: $.id
  amount: $.amount_usd
  account_id: $.user_id

performance:
  timeout_ms: 150
  fallback_score: 0.5
  fallback_decision: flag

Registros de Auditoría

Cada decisión de calificación se escribe automáticamente en un registro de auditoría inmutable y firmado criptográficamente. Los registros son consultables a través de API, exportables a tu SIEM y se retienen durante 7 años por defecto.

GET /v2/audit/events Consulta el registro de eventos de auditoría
ParámetroTipoDescripción
fromdateInicio del rango de fechas (ISO 8601)
todateFin del rango de fechas (ISO 8601)
decisionstringFiltrar por approve, flag, o decline
account_idstringFiltrar por cuenta
rule_hitstringFiltrar eventos donde se activó una regla específica
